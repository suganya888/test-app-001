Simple Calculator App

This is a minimalist, single-page web application that provides basic calculator functionality. It's built with HTML, CSS, and JavaScript, designed to be clean, responsive, and easy to use.

Features:

  • Basic arithmetic operations: Addition, Subtraction, Multiplication, Division.
  • Clear button to reset the calculator.
  • Decimal point support.
  • Visually clean and professional interface.
  • Responsive design for various screen sizes.
  • Basic error handling for invalid operations (e.g., division by zero).

How to Use:

  1. Enter numbers using the number buttons.
  2. Select an operator (+, -, *, /) for your calculation.
  3. Enter the second number.
  4. Press the "=" button to see the result.
  5. Use the "C" button to clear the display and reset the calculator.

Technical Details:

  • Single HTML file with embedded CSS and JavaScript.
  • Utilizes Bootstrap 5 for basic styling and responsiveness (via CDN).
  • Pure JavaScript for calculator logic and user interaction.
