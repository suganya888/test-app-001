README: Simple Calculator App

Simple Calculator App

This is a minimal, single-page web application that functions as a basic calculator. It's built with HTML, CSS, and JavaScript, leveraging Bootstrap for a clean and professional look.

Features:

  • Basic arithmetic operations: Addition, Subtraction, Multiplication, Division.
  • Clear (C) button to reset the calculator.
  • Decimal point support.
  • Simple and intuitive user interface.
  • Responsive design (thanks to Bootstrap and custom CSS).

How to Use:

  1. Enter the first number using the number buttons.
  2. Click an operator button (+, -, *, /).
  3. Enter the second number.
  4. Click the "=" button to see the result.
  5. Use the "C" button to clear the display and reset the calculation.

Development Notes:

  • Single HTML file with embedded CSS and JavaScript.
  • Uses CDN for Bootstrap for external libraries.
  • Includes basic error handling (e.g., division by zero).
  • Designed to be deployable directly to GitHub Pages.

Enjoy calculating!
